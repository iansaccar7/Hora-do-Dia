# Hora do Dia

Este projeto exibe a hora atual do dia e altera a imagem e a saudação de acordo com o período (manhã, tarde ou noite). Ele foi desenvolvido em HTML, CSS e JavaScript.

## Uso
Ao abrir o projeto em um navegador, ele exibirá a hora atual e uma imagem correspondente ao período do dia:

Manhã: Exibe uma imagem de manhã com uma saudação de "Bom Dia".
Tarde: Exibe uma imagem de tarde com uma saudação de "Boa Tarde".
Noite: Exibe uma imagem de noite com uma saudação de "Boa Noite".

## Tecnologias Utilizadas
HTML: Para a estrutura da página.
CSS: Para o estilo e design da interface.
JavaScript: Para a lógica de exibição da hora e mudança de conteúdo.
Font Awesome: Para ícones sociais.
